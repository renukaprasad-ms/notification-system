# AI Usage Declaration

AI tools were used during the development of this project for:

- brainstorming implementation approaches
- generating and refining boilerplate code
- improving documentation
- reviewing architecture and scaling tradeoffs
- identifying bugs and suggesting fixes

All generated or assisted code was reviewed, understood, and modified as needed before being kept in the project.

The final submission reflects code and design decisions that can be explained and maintained by the author.

No public repository code was copy-pasted directly into this project.
